Sobre mi

Actualmente estoy creando proyectos con angular para seguir especializandome.

Trabajando en el desarrollo de una aplicación web donde recepcionan productos en nuestro hubs y realizamos el picking para la entrega adaptándolo al horario de los clientes. Usando una metodología agile SCRUM, bajo la organización del backlog en JIRA. Aplicación desarrollada bajo las tecnologías Vue en su versión 3 para el desarrollo de la UI usando Quasar como framework CSS y para la capa de datos usando Firebase en su versión 9. Nos encargábamos de realizar la consumición de los datos a través de funciones realizadas en Firebase proporcionando una arquitectura reactiva al proyecto, usando como repositorio Github.
